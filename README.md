# Rentify Property Management

Rentify is a role-based property management system built with the MERN stack.  
It allows admins and users to manage properties with different levels of access and also supports buyer inquiry workflow.

## Features

- User Authentication
- Role Based Access Control
- Admin and User Permissions
- Property Add, Update, Delete
- User Account Deactivation
- Buyer Inquiry Workflow
- Admin Dashboard Logic

## Tech Stack

Frontend
- React.js
- CSS Modules

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Roles

### Admin
- Add, update and delete properties
- Manage users
- Deactivate user accounts
- View inquiries and property workflow

### User
- Add properties
- View all properties
- Limited access based on role

## Workflow

A user can add a property to the system.  
When a buyer is interested in a property, they can submit an inquiry form.  
The admin can then view:

- Property details
- Buyer details
- The user/agent who added the property

This helps the admin coordinate communication between both sides.
